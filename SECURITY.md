# Security Policy

## Supported Versions

The latest released version of semver-bump receives security fixes.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately so we can
address it before public disclosure.

- Use GitHub's private [Report a vulnerability](https://docs.github.com/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)
  feature on this repository, **or**
- email the maintainers at `security@example.com`.

Please include steps to reproduce and the affected version. We aim to
acknowledge reports within 5 business days and to share a fix or mitigation
timeline after triage.

Please do **not** open a public issue for security-sensitive reports.
