# Code of Conduct

This project adopts the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, welcoming, and considerate. Harassment or
exclusionary behavior is not tolerated.

## Reporting

To report unacceptable behavior, contact the maintainers at
`conduct@example.com`. All reports will be handled confidentially.
